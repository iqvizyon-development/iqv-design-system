## Install

Iqvizyon UI should be installed as a `dependency` of your app.

```sh
yarn add @iqvizyonui/react-components
```

## Setup

Iqvizyon UI components are styled using CSS in JS. This technique requires a style renderer which inserts CSS into DOM when needed. React context is used to provide the style renderer.

Place a `<IqvizyonProvider />` at the root of your app and pass theme as a prop.

### React 18

```jsx
import React from 'react';
import { createRoot } from 'react-dom/client';
import { IqvizyonProvider, webLightTheme } from '@iqvizyonui/react-components';

import App from './App';

const root = createRoot(document.getElementById('root'));

root.render(
  <IqvizyonProvider theme={webLightTheme}>
    <App />
  </IqvizyonProvider>,
);
```

### React 17

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import { IqvizyonProvider, webLightTheme } from '@iqvizyonui/react-components';

import App from './App';

ReactDOM.render(
  <IqvizyonProvider theme={webLightTheme}>
    <App />
  </IqvizyonProvider>,
  document.getElementById('root'),
);
```

## Usage

That's it. You can now use Iqvizyon UI components in your app.

```jsx
import React from 'react';
import { Button } from '@iqvizyonui/react-components';

export default () => <Button appearance="primary">Get started</Button>;
```

### Strict mode

We are aware of some strict mode bugs when using Iqvizyon UI in React 18. These bugs only show up in strict mode, and they will not stop the rest of your app from running.
You can [track the bugs on Github](https://github.com/iqvizyon-development/iqv-design-system/issues?q=is%3Aopen+is%3Aissue+label%3A%22Area%3A+Strict+Mode%22+label%3A%22React+18%22) and learn how they will affect your application.

#### SSR with Next.js

To avoid strict mode hydration issues, you can disable strict mode in your Next.js app by adding the following configuration to your `next.config.js` file:

```js
module.exports = {
  reactStrictMode: false,
};
```

## Kurulum

Iqvizyon UI, uygulamanızın bir `dependency` olarak kurulmalıdır.

```sh
yarn add @iqvizyonui/react-components
```

## Yapılandırma

Iqvizyon UI bileşenleri CSS-in-JS kullanılarak stillendirilir. Bu teknik, gerektiğinde CSS'i DOM'a ekleyen bir style renderer gerektirir. Style renderer'ı sağlamak için React context kullanılır.

Uygulamanızın kök dizinine bir `<IqvizyonProvider />` yerleştirin ve tema bilgisini prop olarak geçirin.

### React 18

```jsx
import React from 'react';
import { createRoot } from 'react-dom/client';
import { IqvizyonProvider, webLightTheme } from '@iqvizyonui/react-components';

import App from './App';

const root = createRoot(document.getElementById('root'));

root.render(
  <IqvizyonProvider theme={webLightTheme}>
    <App />
  </IqvizyonProvider>,
);
```

### React 17

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import { IqvizyonProvider, webLightTheme } from '@iqvizyonui/react-components';

import App from './App';

ReactDOM.render(
  <IqvizyonProvider theme={webLightTheme}>
    <App />
  </IqvizyonProvider>,
  document.getElementById('root'),
);
```

## Kullanım

İşte bu kadar. Artık uygulamanızda Iqvizyon UI bileşenlerini kullanmaya başlayabilirsiniz.

```jsx
import React from 'react';
import { Button } from '@iqvizyonui/react-components';

export default () => <Button appearance="primary">Başlayın</Button>;
```

### Strict mode

React 18'de Iqvizyon UI kullanırken bazı strict mode hatalarının farkındayız. Bu hatalar yalnızca strict mode'da görünür ve uygulamanızın geri kalanının çalışmasını engellemez.
[Hataları Github üzerinden takip edebilir](https://github.com/iqvizyon-development/iqv-design-system/issues?q=is%3Aopen+is%3Aissue+label%3A%22Area%3A+Strict+Mode%22+label%3A%22React+18%22) ve uygulamanızı nasıl etkileyeceğini öğrenebilirsiniz.

#### Next.js ile SSR

Strict mode hydration sorunlarını önlemek için, `next.config.js` dosyanıza aşağıdaki yapılandırmayı ekleyerek Next.js uygulamanızda strict mode'u devre dışı bırakabilirsiniz:

```js
module.exports = {
  reactStrictMode: false,
};
```
