<div className="iqv-text">

Iqvizyon UI React Components is a set of modern, accessible React UI components built on the Iqvizyon Design System.

Each component is designed to adhere to the following standards:

- **Customizable**: Iqvizyon-styled components by default, but easy to integrate your own brand and theme
- **Performance**: Optimized for render performance
- **Bundle size**: Refactored and slimmed down components that allow you to include the packages and dependencies you need
- **Accessibility**: WCAG 2.1 compliant and tested by trusted testers
- **Design to Code**: Stay up to date with Iqvizyon Design Language changes via Design Tokens

<h3 className="fluent">Genel Bakış</h3>
<div className="iqv-text">

Iqvizyon UI React Components, Iqvizyon Tasarım Sistemi üzerine kurulmuş modern ve erişilebilir React kullanıcı arayüzü bileşenleri setidir.

Her bileşen aşağıdaki standartlara uyacak şekilde tasarlanmıştır:

- **Özelleştirilebilir**: Varsayılan olarak Iqvizyon tarzında tasarlanmıştır, ancak kendi markanızı ve temanızı entegre etmek kolaydır
- **Performans**: İşleme (render) performansı için optimize edilmiştir
- **Paket boyutu**: Yalnızca ihtiyacınız olan paketleri ve bağımlılıkları dahil etmenize olanak tanıyan, yeniden yapılandırılmış ve küçültülmüş bileşenler
- **Erişilebilirlik**: WCAG 2.1 uyumlu ve güvenilir test uzmanları tarafından test edilmiştir
- **Tasarımdan Koda**: Design Tokens aracılığıyla Iqvizyon Design Language değişiklikleriyle güncel kalın

</div>

</Tr>
