<h3 className="fluent">Genel Bakış</h3>
<div className="iqv-text">

Iqvizyon UI React Bileşenleri, Iqvizyon Tasarım Sistemi üzerine kurulmuş modern ve erişilebilir React kullanıcı arayüzü bileşenleri setidir.

Her bileşen aşağıdaki standartlara uyacak şekilde tasarlanmıştır:

- **Özelleştirilebilir**: Varsayılan olarak Iqvizyon tarzında tasarlanmıştır, ancak kendi markanızı ve temanızı entegre etmek kolaydır
- **Performans**: İşleme (render) performansı için optimize edilmiştir
- **Paket boyutu**: Yalnızca ihtiyacınız olan paketleri ve bağımlılıkları dahil etmenize olanak tanıyan, yeniden yapılandırılmış ve küçültülmüş bileşenler
- **Erişilebilirlik**: WCAG 2.1 uyumlu ve güvenilir test uzmanları tarafından test edilmiştir
- **Tasarımdan Koda**: Tasarım Belirteçleri (Design Tokens) aracılığıyla Iqvizyon Tasarım Dili değişiklikleriyle güncel kalın

</div>
