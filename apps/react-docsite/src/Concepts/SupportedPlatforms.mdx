# Supported Platforms

Learn about the platforms that are supported by Iqvizyon UI React.

## Browser

Iqvizyon UI supports the latest, stable releases of all major browsers and platforms.

Please refer to the [Browser Support Matrix](/docs/concepts-developer-browser-support-matrix--docs) for more details.

## React

Iqvizyon UI fully supports React versions 17, 18 and 19.

Please refer to the [React Version Support](/docs/concepts-developer-react-version-support--docs) for more details.

## TypeScript

Iqvizyon UI supports the three latest **major** versions of TypeScript.

> Older TypeScript versions might still work, but they are outside the supported range.

Please refer to the [component patterns guide](https://github.com/iBz-04/iqvui/blob/master/docs/architecture/component-patterns.md) for TypeScript conventions used across Iqvizyon UI React packages.
