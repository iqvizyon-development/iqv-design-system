## Genel Bakış

Iqvizyon UI React grafikleri, Iqvizyon Tasarım Sistemi üzerine kurulmuş modern, erişilebilir, etkileşimli, hafif ve son derece özelleştirilebilir görselleştirme bileşenleri setidir.

Kütüphane, D3 (Data Driven Documents) ve Iqvizyon UI React Bileşenleri kullanılarak oluşturulmuştur.

## Başlarken

Projenizde grafik paketini kullanmaya başlamak için:

1. İstediğiniz paketi `npm` veya `yarn` kullanarak kurun:

   ```
   npm install @iqvizyonui/react-charts
   ```
