## Overview

Iqvizyon UI React charts is a set of modern, accessible, interactive, lightweight and highly customizable visualization components built on the Iqvizyon Design System.

The library is built using D3 (Data Driven Documents) and Iqvizyon UI React Components.

## Getting Started

To get started with the charts package in your project:

1. Install the desired package using `npm` or `yarn`:

   ```
   npm install @iqvizyonui/react-charts
   ```

Iqvizyon UI React grafikleri, Iqvizyon Tasarım Sistemi üzerine kurulmuş modern, erişilebilir, etkileşimli, hafif ve son derece özelleştirilebilir görselleştirme bileşenleri setidir.
