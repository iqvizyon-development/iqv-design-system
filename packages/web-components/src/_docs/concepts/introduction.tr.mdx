## Genel Bakış

Iqvizyon UI Web Bileşenleri, web standartları ve Iqvizyon Tasarım Sistemi ile oluşturulmuş modern ve erişilebilir kullanıcı arayüzü bileşenleri setidir.

Her bileşen aşağıdaki standartlara uyacak şekilde tasarlanmıştır:

- **Özelleştirilebilir**: Varsayılan olarak Iqvizyon tarzında tasarlanmıştır, ancak kendi markanızı ve temanızı entegre etmek kolaydır
- **Performans**: Performans için optimize edilmiştir
- **Paket boyutu**: Yalnızca ihtiyacınız olan paketleri ve bağımlılıkları dahil etmenize olanak tanıyan, yeniden yapılandırılmış ve küçültülmüş bileşenler
- **Birlikte Çalışabilirlik**: Herhangi bir modern JavaScript çerçevesi (framework) ile çalışır
- **Erişilebilirlik**: WCAG 2.2 uyumlu ve güvenilir test uzmanları tarafından test edilmiştir
- **Tasarımdan Koda**: Tasarım Belirteçleri (Design Tokens) aracılığıyla Iqvizyon Tasarım Dili değişiklikleriyle güncel kalın

---

### Web Bileşenleri (Web Components) Nedir?

"Web Bileşenleri", özel HTML öğelerinin oluşturulmasını sağlamaya odaklanan bir web standartları koleksiyonunu ifade eden şemsiye bir terimdir. Bu şemsiye altındaki standartlardan bazıları; yeni HTML etiketleri tanımlama, standart bir bileşen yaşam döngüsüne bağlanma, HTML işleme ve CSS'i kapsülleme (encapsulate), CSS'i parametreleştirme, bileşenleri giydirme (skinning) ve daha fazlasını içerir. Bu platform özelliklerinin her biri W3C tarafından tanımlanmıştır ve bugün her büyük tarayıcıda yerleşik olarak sunulmaktadır.

### Iqvizyon UI Web Bileşenleri, Web Bileşenlerinden Nasıl Yararlanır?

Iqvizyon UI Web Bileşenleri, doğrudan W3C Web Bileşeni standartları üzerine kurulmuştur ve kendi bileşen modelini oluşturmaz. Bu, bileşenlerimizin yerleşik, yerel HTML öğeleriyle aynı şekilde çalışmasını sağlar. Iqvizyon UI bileşenlerini kullanmak için bir çerçeveye (framework) ihtiyacınız yoktur, ancak bunları istediğiniz herhangi bir çerçeve veya kütüphane ile birlikte kullanabilirsiniz.
