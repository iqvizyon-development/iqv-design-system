<h1 class="iqv">Iqvizyon UI Web Components</h1>

## Overview

Iqvizyon UI Web Components is a set of modern, accessible UI components built with web standards and the Iqvizyon Design System.

Each component is designed to adhere to the following standards:

- **Customizable**: Iqvizyon-styled components by default, but easy to integrate your own brand and theme
- **Performance**: Optimized for performance
- **Bundle size**: Refactored and slimmed down components that allow you to include the packages and dependencies you need
- **Interoperability**: Works with any modern JavaScript framework
- **Accessibility**: WCAG 2.2 compliant and tested by trusted testers
- **Design to Code**: Stay up to date with Iqvizyon Design Language changes via Design Tokens

---

### What are Web Components?

"Web Components" is an umbrella term that refers to a collection of web standards focused on enabling the creation of custom HTML elements. Some of the standards that are under the umbrella include the ability to define new HTML tags, plug into a standard component lifecycle, encapsulate HTML rendering and CSS, parameterize CSS, skin components, and more. Each of these platform features is defined by the W3C and has shipped in every major browser today.

### How does Iqvizyon UI Web Components leverage Web Components?

Iqvizyon UI Web Components is built directly on the W3C Web Component standards, and does not create its own component model. This allows our components to function the same as built-in, native HTML elements. You do not need a framework to use Iqvizyon UI components but you can use them in combination with any framework or library of your choice.

## Genel Bakış

Iqvizyon UI Web Components, web standartları ve Iqvizyon Tasarım Sistemi ile oluşturulmuş modern ve erişilebilir kullanıcı arayüzü bileşenleri setidir.

Her bileşen aşağıdaki standartlara uyacak şekilde tasarlanmıştır:

- **Özelleştirilebilir**: Varsayılan olarak Iqvizyon tarzında tasarlanmıştır, ancak kendi markanızı ve temanızı entegre etmek kolaydır
- **Performans**: Performans için optimize edilmiştir
- **Paket boyutu**: Yalnızca ihtiyacınız olan paketleri ve bağımlılıkları dahil etmenize olanak tanıyan, yeniden yapılandırılmış ve küçültülmüş bileşenler
- **Birlikte Çalışabilirlik**: Herhangi bir modern JavaScript framework ile çalışır
- **Erişilebilirlik**: WCAG 2.2 uyumlu ve güvenilir test uzmanları tarafından test edilmiştir
- **Tasarımdan Koda**: Design Tokens aracılığıyla Iqvizyon Design Language değişiklikleriyle güncel kalın

---

### Web Components Nedir?

"Web Components", özel HTML öğelerinin oluşturulmasını sağlamaya odaklanan bir web standartları koleksiyonunu ifade eden şemsiye bir terimdir. Bu şemsiye altındaki standartlardan bazıları; yeni HTML etiketleri tanımlama, standart bir bileşen yaşam döngüsüne bağlanma, HTML işleme ve CSS'i kapsülleme, CSS'i parametreleştirme, bileşenleri giydirme ve daha fazlasını içerir. Bu platform özelliklerinin her biri W3C tarafından tanımlanmıştır ve bugün her büyük tarayıcıda yerleşik olarak sunulmaktadır.

### Iqvizyon UI Web Components, Web Components'ten Nasıl Yararlanır?

Iqvizyon UI Web Components, doğrudan W3C Web Component standartları üzerine kurulmuştur ve kendi bileşen modelini oluşturmaz. Bu, bileşenlerimizin yerleşik, yerel HTML öğeleriyle aynı şekilde çalışmasını sağlar. Iqvizyon UI bileşenlerini kullanmak için bir framework'e ihtiyacınız yoktur, ancak bunları istediğiniz herhangi bir framework veya kütüphane ile birlikte kullanabilirsiniz.
