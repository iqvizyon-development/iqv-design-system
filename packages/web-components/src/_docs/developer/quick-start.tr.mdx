# Hızlı Başlangıç

## Iqvizyon UI Web Bileşenleri v3 Kurulumu

Iqvizyon UI web bileşenleri ve Iqvizyon belirteçleri (tokens), uygulamanızın bir bağımlılığı (`dependency`) olarak kurulmalıdır. Her iki paket de [unpkg üzerinde mevcuttur](https://unpkg.com/@iqvizyonui/web-components).

```sh
npm install @iqvizyonui/web-components @iqvizyonui/tokens
```

## Temel Kullanım

Iqvizyon UI Web Bileşenleri, CSS değişkenleri biçimindeki belirteçler (tokens) kullanılarak stillendirilir. Web siteniz veya uygulamanız için bir tema sağlamak üzere `setTheme` yardımcı işlevini kullanabilirsiniz.

Mevcut temalar:

- `webLightTheme`
- `webDarkTheme`

Tema uygulaması:

```js
import { setTheme } from '@iqvizyonui/web-components';
import { webLightTheme } from '@iqvizyonui/tokens';

setTheme(webLightTheme);
```

### Tüm bileşen paketini içe aktarma

Iqvizyon Web Bileşenleri ile hızlıca çalışmaya başlamanın en kolay yolu, tüm bileşen paketimizi kullanmaktır. Ayrıca, stil sayfasını kendiniz oluşturarak veya `setTheme` yardımcı işlevimizi kullanarak Iqvizyon UI'ın tasarım belirteci CSS'ini sayfaya eklemeniz gerekir.

```js
import '@iqvizyonui/web-components/web-components.js';
```

### Önerilen Kullanım: Ağaç sallama (tree-shaking) özellikli bileşen dışa aktarımlarımızı kullanın

Üretim paketi boyutunuzu azaltmak için her bileşeni bağımsız bir ESM dışa aktarımı olarak sunuyoruz. Maksimum performans ve ağaç sallama (tree-shaking) için bu yöntemi öneriyoruz. Bağımsız bileşenleri içe aktarmak için iki seçenek vardır.

### Tanımlanmış bileşeni içe aktarma

Tanımlanmış bileşen dışa aktarımlarımız, tekil bileşenleri içe aktarır ve sayfadaki özel öğeyi (custom element) tanımlar.

```js
import '@iqvizyonui/web-components/button.js';
```

Not: Bir bileşenin alt bileşen bağımlılığı varsa, her iki bileşeni de içe aktarmanız gerekir (örneğin, Tablist ve Tab).

### Adlandırılmış içe aktarmaları kullanarak öğeyi kendiniz tanımlama

Yan etkileri (side-effects) kontrol etmek istiyorsanız ve bileşen tanımlarınızın ne zaman tanımlanacağı üzerinde ayrıntılı kontrole ihtiyacınız varsa, her bileşen tanımı için adlandırılmış içe aktarmaları içe aktarın ve `define` yöntemini çağırın.

```js
import { Button } from '@iqvizyonui/web-components/button/class.js';
import { definition as buttonDefinition } from '@iqvizyonui/web-components/button/definition.js';

Button.define(buttonDefinition);
```
