# @iqvizyonui/react-popover

**React Popover components for [Iqvizyon UI React](https://iqvizyon-development.github.io/iqv-design-system/react/)**

To import React Popover components:

```js
import { Popover, PopoverTrigger, PopoverSurface } from '@iqvizyonui/react-popover';

<Popover>
  <PopoverTrigger>
    <button>Controls popover</button>
  </PopoverTrigger>

  <PopoverSurface>
    <div className={example}>
      <h3>Popover content</h3>

      <div>This is some popover content</div>
    </div>
  </PopoverSurface>
</Popover>;
```
