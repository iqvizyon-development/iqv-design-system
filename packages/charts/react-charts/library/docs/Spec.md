# @iqvizyonui/react-charts Spec

## Background

Iqvizyon UI React charts is a set of modern, accessible, interactive, lightweight and highly customizable visualization components built on the Iqvizyon Design System.

The library is built using D3 (Data Driven Documents) and the Iqvizyon UI design system.
